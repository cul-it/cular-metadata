Draft elements for a local standard

| Element | Description |
|---------| ----------- |
| Package BibID | Bibliographic record ID this package is associated with. |
| Package Barcode | Barcode assigned to the physical manifestation or the physical media carrier of the assets referenced in this package. | 
| Package Media Number | Media Number assigned to the physical manifestation or the physical media carrier of the assets referenced in this package. (Primarily associated with the RMC Media Number.) |
